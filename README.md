# India_HeartAge_DrSuprakash

India Heart Age Calculator — mobile-optimized

This repository contains a single-file, client-side HTML heart-age calculator tailored for the Indian population. It provides non-laboratory and optional lab-based heart age estimates using BMI or waist-based thresholds.

## Files
- `index.html` — the mobile-optimized calculator (open in any browser).
- `LICENSE` — MIT license.
- `CITATION.cff` — citation information.
- `README.md` — this file.

## Publish to GitHub Pages
1. Create a new repository on GitHub called `India_HeartAge_DrSuprakash` under your account `drsuprakashcm-lgtm`.
2. Upload these files to the repository root.
3. In the repository, go to **Settings → Pages**, set **Branch** to `main` and **Folder** to `/ (root)` and save.
4. Your site will be available at: `https://drsuprakashcm-lgtm.github.io/India_HeartAge_DrSuprakash/` (it may take a minute to publish).

## Notes
- This tool is for screening and research; not a clinical diagnostic.
- Edit `index.html` to tweak thresholds, labels, or translations.
